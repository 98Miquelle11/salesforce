### [Lightning Web Components Basics](https://trailhead.salesforce.com/content/learn/modules/lightning-web-components-basics)
This folder contains a project (**bikeCard**) authorized to a Salesforce org (which was created for exercises within the following module). This project was made for versioning and development of metadata (JavaScript, HTML, Apex etc.).

1. [Discover Lightning Web Components](https://trailhead.salesforce.com/content/learn/modules/lightning-web-components-basics/discover-lightning-web-components?trail_id=force_com_dev_beginner)

   * Explanation of the Lightning Web Components programming model (HTML, ES6+ JavaScript, CSS),
   * List the benefits of using Lightning Web Components (high performance, web standards aligned, developer productivity, full encapsulation, backward compatibility, efficient tooling),
   * What is needed to get started developing Lightning Web Components (DevOps Center, Code Builder, Dev Hub and Scratch Orgs, GitHub).

2. [Create Lightning Web Components](https://trailhead.salesforce.com/content/learn/modules/lightning-web-components-basics/create-lightning-web-components)

   * How to build and structure Lightning Web Components,
   * 3 main files of components (`.html`, `.js`, `.css`),
   * JavaScript features - decorators (@api, @track, @wire), lifecycle hooks (e.g. connectedCallback()),
   * Naming conventions (camelCase, kebab-case).

3. [Deploy Lightning Web Component Files](https://trailhead.salesforce.com/content/learn/modules/lightning-web-components-basics/push-lightning-web-component-files)

   * What LWC consists (`.html` template markup, `.js` component logic, `.js-meta.xml` visibility/config for Lightning pages),
   * How to deploy Lightning Web Component Files.

4. [Handle Events in Lightning Web Components](https://trailhead.salesforce.com/content/learn/modules/lightning-web-components-basics/handle-events-in-lightning-web-components)

   * Event handling in LWC,
   * Creating and dispatching custom events,
   * Listening to custom events,
   * Best practices.

5. [Add Styles and Data to a Lightning Web Component](https://trailhead.salesforce.com/content/learn/modules/lightning-web-components-basics/add-styles-and-data-to-a-lightning-web-component)

   * Property binding,
   * The Wire Service.
